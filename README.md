🔐 Password Generator Web App

A simple and secure Password Generator Web Application built using HTML, CSS, and JavaScript.
This tool helps users create strong and random passwords with customizable options.

📌 Features

Generate secure random passwords

Adjustable password length slider

Option to include:

    Uppercase letters

    Lowercase letters

    Numbers

    Symbols

Password strength indicator (Weak / Medium / Strong)

Copy password button for easy clipboard copying

Clean and responsive user interface

🛠️ Technologies Used

HTML5 – Structure of the webpage

CSS3 – Styling and layout

JavaScript – Password generation logic and interactivity

⚙️ How It Works

Select the password length using the slider.

Choose the character types:

Uppercase

Lowercase

Numbers

Symbols

Click Generate Password.

The generated password will appear in the output box.

Click Copy to copy the password to your clipboard.

💻 Example Output
Generated Password: A7@kLp!9Zx3Q
Password Strength: Strong
